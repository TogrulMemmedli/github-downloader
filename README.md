# 📦 GitHub Downloader

**GitHub Downloader** is a powerful yet lightweight Chrome Extension that adds a dedicated download button next to every file and folder in the GitHub repository interface. No more cloning massive repositories just to get a single folder!

## ✨ Features
* **📁 Folder Download**: Instantly download any directory as a `.zip` archive.
* **📄 File Download**: Download individual files with one click.
* **🔒 Private Repository Support**: Full access to your private projects using a Personal Access Token (PAT).
* **⚡ Fast & Secure**: Direct integration with the GitHub API. Your token stays on your device and is never shared.

## 🛠️ How to Install (Manual Setup)
Since the extension is currently in development, you can install it manually:

1. **Download** this repository as a ZIP and extract it (or clone it).
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **"Developer mode"** (toggle in the top right corner).
4. Click **"Load unpacked"** and select the folder where you extracted the files.

## 🚀 Getting Started
To use the extension with private repos or to avoid API rate limits:

1. Create a [GitHub Personal Access Token (classic)](https://github.com/settings/tokens) with the `repo` scope.
2. Click the **GitHub Downloader** icon in your browser's extension bar (puzzle icon).
3. Save your token in the settings.
4. Refresh your GitHub page and start downloading!
